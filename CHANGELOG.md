# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project follows Semantic Versioning.

## [1.0.0] - 2026-04-14

### Added
- Initial stable public release of the FirecastRL wildfire suppression environment.
- Gymnasium-compatible wildfire environment with human, `rgb_array`, and packaged 3D browser rendering modes.
- Helitack agent dynamics for aerial fire suppressant drops.
- Observation and reward wrappers for RL experimentation.
- Packaged 3D viewer assets and example runner scripts.

### Documentation
- Updated README with `uv`-based setup, renderer usage, screenshots, citation details, and troubleshooting guidance.
